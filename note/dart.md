# dart笔记

## 易错

```
??：表示当一个值为空时，给他赋值后面的数据  expr1??expr2;如果expr1为非空，则返回其值，否则返回expr2的值
?.：条件成员访问运算符，做一次判空操作，  optionalThing?.isEnabled??false;
...?
??=
.. （级联运算符）
=> : 只包含一个表达式的函数，可以使用=>
final和const:final不用先赋值，const声明时必须赋值，不然会报错。
```
